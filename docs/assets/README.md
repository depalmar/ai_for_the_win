# Documentation Assets

This folder contains visual assets for the AI for the Win documentation and GitHub landing pages.

## Contents

| File | Description | Used In |
|------|-------------|---------|
| `banner.svg` | Main project banner with logo and stats | README.md, docs/index.md |

## Guidelines

- Use SVG format for logos and banners (scalable, small file size)
- Keep colors consistent with project theme (see docs/index.md CSS variables)
- Optimize images before committing

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Primary | `#6366f1` | Buttons, highlights |
| Secondary | `#10b981` | Success, ML labs |
| Accent | `#f59e0b` | Warnings, advanced labs |
| Danger | `#ef4444` | Expert labs, alerts |
| Background Dark | `#0d1117` | Page background |
| Background Card | `#161b22` | Card backgrounds |
| Text | `#c9d1d9` | Primary text |
| Text Muted | `#8b949e` | Secondary text |
| Border | `#30363d` | Borders, separators |
